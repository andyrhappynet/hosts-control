# Hosts Control

## 📋 Sistema de control de acceso web por grupos

### Grupos disponibles:
- **Ventas** - Restricción máxima
- **Retencion** - Restricción moderada  
- **Administracion** - Restricción mínima
- **Calidad** - Restricción media

## 🚀 Instalación
1. Descargar `scripts/instalar_servicio_hosts.bat`
2. Ejecutar como Administrador
3. Seleccionar el grupo del equipo

## 📊 Administración
Editar archivos en carpeta `/grupos/`

Formato: `127.0.0.1 nombre-sitio.com`

## 🔄 Los cambios aplican al reiniciar el equipo
